# Cryptocurrency Charting

> A single-page app for viewing the historical price data for bitcoin, powered by CoinDesk.

## Dependencies

1. React, React-DOM
1. Express
1. Chart.js

## Dev Dependencies

1. Nodemon
1. Webpack, Webpack CLI
1. Babel, Babel Preset-React, Babel Loader
1. Live-Server

## Usage

```sh
npm install
npm run build
npm run start
live-server
```
